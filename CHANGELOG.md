# Changelog

All notable changes to spec2gcp will be documented here.

## [1.0.0] - 2026-03-07

### Added
- Initial release of spec2gcp - Google Cloud variant of spec2cloud
- 10 specialized GitHub Copilot agents (spec2gcp, pm, devlead, architect, planner, dev, gcloud, tech-analyst, modernizer, extender)
- 12 workflow prompts (prd, frd, plan, implement, delegate, deploy, rev-eng, modernize, extend, adr, bootstrap-agents, generate-agents)
- MCP server configuration for GCP, observability, storage, analytics, context7, github, playwright, deepwiki, developer-knowledge
- Dev container with Python 3.12, gcloud CLI, Node.js, Docker-in-Docker
- APM (Agent Package Manager) integration with spec2gcp-guidelines packages
- MkDocs documentation site with Material theme
- GitHub Actions release workflow (full and minimal packages)
- Installation scripts for Linux/Mac (install.sh, quick-install.sh) and Windows (install.ps1)
